
# BMIMIC

## Overview

This package is for generating example simulation data for BMIMIC model
which is introduced in **paper**. More information can be found in
**paper**.

## Install

``` r
devtools::install_github("sooyongl/BMIMIC")
```

## Generate Data

``` r
library(BMIMIC)

sdat <- makeDat()
```
